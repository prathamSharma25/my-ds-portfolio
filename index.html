Hurricane Damage Prediction using CNNs

  • Project Category: Computer Vision & Image Processing

In this project, a combination of an image processing algorithm (which makes use of the Canny edge detection, and Gaussian blur methods) and a custom Convolutional Neural Network model is used to identify areas that have been damaged by hurricanes through satellite imagery.

The custom CNN model achieves an accuracy of 93% on the test set. The custom CNN model also achieves a 25% reduction in model complexity as compared to the pre-trained VGG-16 model, with minimal difference in model performance.

Word Embedding Based Answer Evaluation System

  • Project Category: AI Smart System

WebAES (Word Embedding Based Answer Evaluation System) is an AI expert system that enables the automatic evaluation of online subjective tests. Using a smart word embedding based algorithm (implemented using the Doc2Vec and BERT models), WebAES evaluates the answer responses submitted by students and awards appropriate marks based on the answer expected by the faculty for the same question.

Faculties can create questions with the expected answer and create tests using these questions. Students can take available tests and submit their answers to WebAES. WebAES then evaluates the submitted answers and gives back the results in minutes.

This helps in saving a lot of time and effort for faculties in evaluating assessments by automating the entire process of answer evaluation. WebAES saves time and effort while maintaining reliability in answer evaluation for online assessments.

Spotify EDA & Recommendation System

  • Project Category: Recommendation System

In this project, I have developed a Content-based recommendation System for Spotify. For the same, I have used a dataset of 600,000 tracks and 1.1M artists on Spotify, available on Kaggle. Based on the song given, the recommendation system recommends more songs as well as artists that the user might also like.

The recommendation system uses information describing each song/track, such as its audio characteristcs (instrumentalness, tempo, etc.) and genre information, to determine other similar songs. The same is done for artists as well. Using the Vector Space Model, each song and artist is represented as a vector. Similarity is then determined using cosine similarity measure. The most similar songs and artists are recommended.

This tweets dataset was collected by scraping Twitter for tweets that contain the hashtag "#FarmersProtest", using the Python library snscrape. This project is carried out in 2 parts. In the first part, I have performed some data pre-processing and transformation on the raw data.
